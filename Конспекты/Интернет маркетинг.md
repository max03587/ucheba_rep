### Что такое реклама в интернете?
Где бывает:
- Поисковые системы
- социальные сети
- мессенджеры
- сайты
- видеоплатформы
- мобильные приложения

### Почему интернет реклама важна?
- Люди ежедневно находятся в интернете
- Бизнес может быстро находить клиентов
- Рекламу можно точно настраивать
- Результат можно точно измерять
- Компании можно быстро менять и развивать

### Отличия интернет рекламы от традиционной

| Традиционная реклама        | Интернет-реклама                      |
| --------------------------- | ------------------------------------- |
| Массовый охват              | Точный таргетинк                      |
| Сложно менять полсе запуска | Можно быстро редактировать            |
| Трудно измерить результаты  | Почти все можно посчитать             |
| Дорого для малого бизнеса   | можно стартовать с небольшого бюджета |
|                             |                                       |
Креатив - изображение, видео с целью привлечь клиента
### Главные задачи интернет-рекламы
Интернет реклама нужна, чтобы:
- привлечь внимание
- сформировать желание
- напомнить о бренде
- вызвать интерес
- подтолкнуть к действию
- вернуть аудиторию повторно

### Где можно встретить интернет-рекламу
- в Google и Яндекс
- на Youtube и других видеоплатформах
- в мобильных приложения
- у блогеров
- в социальных сетях
- на новостных и тематических сайтах
- в онлайн-магазинах
- в мессенджерах

### Основные виды интернет-рекламы
- Поисковая реклама
- Таргетированная реклама
- Медийная реклама
- Видеореклама
- Нативная реклама
- Ретаргетинк/ремаркетинг

### Поисковая реклама
Это объявления, которые показываются в ответ на запрос пользователя.
- Работает по уже существующему спросу
- Подходит для товаров и услуг, которые люди сами ищут
- часто приводит "горячую" аудиторию

### Таргетированная реклама
- Возраст
- Пол
- Интересы
- География
- Поведение
- Подписки и активности
Работает в основном только в социальных сетях.

### Медийная реклама
Это визуальная реклама.
- Баннеры
- Изображения
- Анимации
- Графические блоки на сайтах и приложениях

### Видеореклама
- Перед видео
- Внутри видео
- После видео
- В ленте социальных сетей
- На видеоплатформах

### Нативная реклама
Это реклама, встроенная в контент и похожая на него по стилю.
Примеры:
- Статья с интеграцией бренда
- Обзор у блогеров
- Полезный пост с упоминанием продукта
- Рекомендация сервиса внутри контента
Преимущество:
- Производится легче.

### Ретаргетинк/ремаркетинг
Это реклама для пользователей, уже взаимодействовавших с брендом.

### Схема работы интернет-рекламы
Бизнес ставит цель **→** Определяется аудитория **→** Выбирается площадка **→** Создаются объявления **→** Запускается компания **→** Пользователи видят рекламу **→** Совершают действия **→** Результаты анализируются

### Из чего состоит рекламная компания
- Цель
- Целевая аудитория
- Оффер
- Креатив
- Площадка
- Посадочная страница
- Аналитика
