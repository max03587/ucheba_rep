#python 
### Введение

Класс - это пользовательский тип данных, конструкция ООП, которая описывает тот или иной объект, его свойства и способы взаимодействия с ним. Класс имеет поля - переменные, которые будут принадлежать к его экземпляру, а также методы - функции, которые также можно будет использовать для его экземпляра. Простыми словами класс - это шаблон для реализации какого то объекта, например мы знаем как должна выглядеть кружка, какие хар-ки она имеет. При помощи класса мы описываем абстрактную "маму" кружку, а экземпляры будут являться настоящими, конкретными кружками с хар-ми мамы кружки. 
Класс решает несколько важных задач таких как:
- Дублирование кода
- Унификация объектов (т.е. одинаковые по задумке объекты действительно будут иметь одинаковые хар-ки и функционал)

### Классы в python

Объявляются при помощи ключевого слова Class и указания имени класса, затем ставиться двоеточие и со следующей строки описывается тело класса. Чтобы объявить поля класса используется ключевое слово self, после которого, через точку указывается имя поля. Поле это тоже самое что и переменная, но внутри класса. В параметрах метода также должен быть указан self.

```python

class Per:
	def test(self):
		self.msg = "Hello!"
		
```

Для инициализации полей при создании экземпляра класса используется функция конструктор. Она также обязательно принимает параметр self. Эта функция будет автоматически вызвана при создании экземпляра класса. Если она не объявлена, будет использован конструктор. 
```python

class Per:
	def __init__(self, text):
		self.msg = text
		
Per_example = Per('Hello!')
```
