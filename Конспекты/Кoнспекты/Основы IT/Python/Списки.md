### Определение
У порядочные, изменяемые коллекции произвольных типов. Почти как массив, но в отличии от него значения в списках могут изменятся, отличатся. Они создаются с помощью квадратных скобок или функцией - конструктора (List). К каждому элементу можно обратится по его индексу. Индексация начинается с 0. 

Списки не обязательно должны содержать однотипные объекты.

a = []
a = list()
oa = ["pavuk", True, 3]

### Функция LIst

Принимает значения итерируемого типа.

Срезы это способ получения части списка по 3 аргументам. Для создания среза указывается имя списка и в кв. скобочках аргументы. Всего есть 3 способа указать правила генерации срезов. [start, end, step]

Если требуется использовать один из указанных способов, но отсутствует некоторая информация, то можно поставить : вместо пропущенного значения  

### Методы работы со списками

- append - добавить
```python
list1 = [1, 2, 3, 4, 5]
list1.append(6) 
print(list1)
```
- clear - убирает все из списка
```python
list1.clear() 
print(list1)
```
- index - находит по индексу
```python
list2 = [1, 2, 3, 4, 5] 
index_three = list2.index(3) 
print(index_three)
```
- len - возвращает длину списка
```python
length = len(list2) 
print(length)
```
- min - возвращает минимальное значение
```python
minimum = min(list2) 
print(minimum)
```
- max - возвращает максимальное значение
```python
maximum = max(list2) 
print(maximum)
```
- sorted - сортирует список
```python
sorted_list = sorted(list2) 
print(sorted_list)
```
- insert - добавляет на указанное по индексу место
```python
list2.insert(2, 'three') 
print(list2)
```
- extend - добавляет несколько элементов из другого списка (или любой другой коллекции)
```python
list2.extend([6, 7]) 
print(list2)
```
- remove - удаляет первое вхождение элемента по его значению.
```python
list2.remove('three') 
print(list2)
```
- pop - удаляет элемент по индексу
```python
list2.pop(a)
```
- count - позволяет подсчитать кол-во вхождений определённого элемента 
```python
list2.count(a)
```
- copy - создает копию списка, чтобы работать с ней не изменяя оригинал 
```python
list2.copy(a)
```
