Вы задаете вопрос шару в консоли, он отвечает рандомным образом
```python
import random

print("Я магический шар, и я знаю ответ на любой твой вопрос.")

name = input("Как тебя зовут?\n")

print("Привет,", name, end=". ")

answer = [ "Да", "Пока неясно, попробуй снова", "Даже не думай", "Вероятнее всего", "Спроси позже", "Нет", "Никаких сомнений", "Определённо да", "Сейчас нельзя предсказать", "Перспективы не очень хорошие", "Можешь быть уверен в этом", "Сконцентрируйся и спроси опять", "Весьма сомнительно"]

while True:

    question = input("Введи свой вопрос:\n")

    print(random.choice(answer))

    next = input("Хочешь задать еще один вопрос? Введи да или нет.\n")

    if next.lower() == "да":

        continue

    else:

        break

print("Возвращайся если возникнут вопросы!")
```
