Docker - инструмент для контейнеризации, который упаковывает приложения с зависимостями в легкий изолированный контейнер. Он изолирует процессы на уровне ядра и работает на одном хосте. Его задача увеличить совместимость, упростить деплой и воспроизводимость.  

Деплой — это процесс развертывания и запуска готового программного обеспечения на сервере или рабочей платформе, чтобы оно стало доступно пользователям.
Kubernetes - инструмент для управления контейнерами, обеспечивающий высокую эффективность и отказоустойчивость развернутых систем.
