## Наследование Python

Наследование позволяет определить класс, который наследует все методы и свойства другого класса.

**Родительский класс** — это класс, от которого наследуется, также называемый Базовый класс.

**Дочерний класс** — это класс, который наследует от другого класса, также называют производным классом.

---

## Создать родительский класс

Любой класс может быть родительским классом, поэтому синтаксис такой же, как при создании любого Другой класс:

Создаём класс с именем , с и свойствами, и метод:`Person``firstname``lastname``printname`

```python
class Person:  
  def __init__(self, fname, lname):  
    self.firstname = fname  
    self.lastname = lname  
  
  def printname(self):  
    print(self.firstname, self.lastname)  
  
#Use the Person class to create an object, and then execute the printname method:  
  
x = Person("John", "Doe")  
x.printname()

```

## Создать дочерний класс

Чтобы создать класс, который наследует функциональность от другого класса, при создании дочернего класса отправьте родительский класс в качестве параметра Класс:

### Пример

Создайте класс с именем , который унаследует свойства и методы из Класс:`Student``Person`

```python
class Student(Person):  
	pass
```
