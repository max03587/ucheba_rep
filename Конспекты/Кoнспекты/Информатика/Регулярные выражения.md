Регулярные выражения (**regex**) — это мощный инструмент для поиска и обработки текста по шаблону. В Python они реализованы в модуле **_re_**, который поддерживает широкий набор синтаксических конструкций, включая группы, квантификаторы, классы символов и lookaround.
**Основные функции модуля _re_:**

- **_re.search(pattern, string)_** — ищет первое совпадение.
    
- **_re.fullmatch(pattern, string)_** — проверяет, соответствует ли вся строка шаблону.
    
- **_re.findall(pattern, string)_** — возвращает список всех непересекающихся совпадений.
    
- **_re.finditer(pattern, string)_** — возвращает итератор match-объектов.
    
- **_re.split(pattern, string)_** — разбивает строку по шаблону.
    
- **_re.sub(pattern, repl, string)_** — заменяет совпадения на строку или результат функции.

Пример:
```python
import re
text = "Сегодня 19.01.2018, а завтра 20.01.2018"
# Найти все даты
dates = re.findall(r'\d{2}\.\d{2}\.\d{4}', text)
print(dates) # ['19.01.2018', '20.01.2018']
# Замена формата даты
new_text = re.sub(r'(\d{2})\.(\d{2})\.(\d{4})', r'\2/\1/\3', text)
print(new_text) # Сегодня 01/19/2018, а завтра 01/20/2018
