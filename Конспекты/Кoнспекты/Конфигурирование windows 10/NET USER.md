### 1. Просмотр списка пользователей

- Команда:
    
    bash
    
    ```bash
    net user
    ```
    
- Описание:  
    Выводит список всех учетных записей пользователей на локальном компьютере.

### 2. Изменение пароля пользователей

- Команда:
    
    bash
    
    ```bash
    net user <имя_пользователя> <новый_пароль>
    ```
    
- Описание:  
    Устанавливает новый пароль для указанного пользователя. Например,
    
    bash
    
    ```bash
    net user Ivan P@ssw0rd
    ```
    
    изменит пароль пользователя Иван на `P@ssw0rd`.

### 3. Просмотр данных пользователя

- Команда:
    
    bash
    
    ```bash
    net user <имя_пользователя>
    ```
    
- Описание:  
    Показывает подробную информацию о выбранной учетной записи (статус, присвоенные группы, дата последнего входа и др.).

### 4. Просмотр локальных групп

- Команда:
    
    bash
    
    ```bash
    net localgroup
    ```
    
- Описание:  
    Выводит список всех локальных групп на компьютере.

### 5. Назначение пользователя в локальную группу

- Команда:
    
    bash
    
    ```bash
    net localgroup <имя_группы> <имя_пользователя> /add
    ```
    
- Описание:  
    Добавляет указанного пользователя в выбранную группу. Например,
    
    bash
    
    ```bash
    net localgroup Administrators Ivan /add
    ```
    
    добавит пользователя Иван в группу Администраторов.

### 6. Запрет на изменение пароля

- Команда:
    
    bash
    
    ```bash
    net user <имя_пользователя> /passwordchg:no
    ```
    
- Описание:  
    Запрещает пользователю менять свой пароль.

### 7. Удаление пользователя

- Команда:
    
    bash
    
    ```bash
    net user <имя_пользователя> /delete
    ```
    
- Описание:  
    Удаляет указанного пользователя из системы.
